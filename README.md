## Installation and Setup Instruction
#### Run the following commands in your terminal

`git clone https://github.com/Gaursafal/Shopping-Cart`

`npm install`

`npm start`

`npm run api`

## Libraries installed

- Material UI
- Axios
- React router dom
- Styled components
- Express-js
- Cors
